# technical-test-employee-service
Proyecto para el desarrollo de un servicio básico de gestión empleados. La solución incluye:

 - API REST.

 - Servicio SOAP.

 - Base de datos para la persistencia de la información de los empleados.

 - Pruebas unitarias para validar el comportamiento de los componentes principales.

 - Docker para un despliegue simple de la base de datos.

Este repositorio se irá construyendo progresivamente siguiendo buenas prácticas de desarrollo:

 - Uso de ramas feat/type-of-change(docs,rest,soap)/brief-description para cada componente importante.

 - Pull Requests para mantener la trazabilidad del trabajo.

 - Documentación general del proyecto.
